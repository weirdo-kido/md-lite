# ***md* lite**  
simple modern markdown logic on 8kb code. zero dependencies.  
I change the `>` to `/` for quote because I can't force script works for the symbol for quote.  
Read the [code](index.html) from line 219 to 234 for logic.  
Its just a simple markdown, I use experimental ui design for mobile first, its my opiniated ui ux.  
## shortcut
`ctrl` + `.` to open Window, toogle View\Edit after Window Open;   
`Esc` for close Window.   
Double tap or double click the windows header to export to `note.md`
